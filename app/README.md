# WorkFlow Hub — Home Prototype

정적 HTML/CSS/JS로 구현한 Home 화면 인터랙티브 프로토타입.
`manager/` 폴더가 팀장(Manager) 역할 기준 구현체이다.

## 실행

```bash
cd vibe-projects
python3 -m http.server 8787
# http://localhost:8787/manager/ 접속
```

## 데이터 컨텍스트

- **디자인 에이전시 시나리오**: 외부 클라이언트 회사로부터 업무요청을 수신하는 디자인 에이전시 맥락
- **동적 날짜 시프트**: 앵커 날짜(`_ANCHOR`) 기준으로 실행 시점의 오늘 날짜에 맞게 모든 더미 데이터 날짜를 자동 조정

---

## 레이아웃

```
[다크 사이드바 220px] | [메인 콘텐츠]
                           ├─ 헤더 (주간 날짜 필터 / 검색·알림·프로필)
                           └─ 3컬럼 그리드 (equal width)
                               ├─ col1: 이번 주 업무
                               ├─ col2: 오늘 할 일
                               └─ col3: 오늘의 현황 + 업무 요청
```

## 디자인 시스템

- **폰트**: DM Sans (UI) / DM Mono (KPI 숫자)
- **사이드바**: `#0e0e16` 배경, 활성 탭 inset 좌측 보더
- **CSS 변수**: `--bg`, `--surface`, `--surface-muted`, `--line`, `--line-soft`, `--text`, `--muted`, `--soft`, `--blue`, `--blue-soft`, `--shadow`, `--shadow-lg`, `--radius`, `--radius-sm`
- **모달/오버레이**: `backdrop-filter: blur` 적용

## 공통 헤더

- **주간 필터(좌)**: `< 2026 06.08 ~ 06.14 이번 주 >` 형태, 주 단위 이동
- **검색(우)**: 인라인 인풋박스 (포커스 시 확장)
- **알림 팝오버(우)**: 읽음/안읽음 표시, 전체 읽음 처리
  - 알림 아이콘: 미읽 수 뱃지 (99+ 처리), 모두 읽으면 숨김
  - 알림 레벨 배지(긴급/중요/일반) 없음
  - 업무요청 알림: 제목 옆에 업무항목명 파란 칩 표시

### 알림 발생 케이스

| 알림 제목 | 트리거 |
|---|---|
| 업무요청 도착 | 업무요청 전송 시 |
| 업무요청 수락 | 수락 모달 제출 시 |
| 업무요청 거절 | 거절 모달 제출 시 |
| 업무항목 추가 | 회의록 액션아이템 → 내 업무 추가 시 |
| 연차 신청 완료 | 연차 신청 제출 시 |
| 연차 승인 | Manager/Owner가 연차 승인 시 |
| 연차 반려 | Manager/Owner가 연차 반려 시 |
| 회의 등록 완료 | 회의 등록 시 (참석자 수 포함) |

## Home 페이지

### col1: 이번 주 업무 (Weekly Tasks)

- 요일(월~일)별 카드 그룹화, 오늘 카드 파란 배경
- 유형 아이콘: 핀(고정) · 빨간 원(긴급) · 그레이 원(일반)
- 고정 → 긴급 → 일반 순 정렬, 고정업무 아래 구분선
- 지연 배지: 마감일 경과 + 미완료 세션 존재 시 (고정업무 제외)
- **날짜(MM/DD) 클릭** → Daily Todo 날짜 전환
- **업무명 클릭** → Daily Todo에 인라인 입력창 표시
- **편집 버튼** (hover) → 업무 상세 슬라이드 패널
- **삭제 버튼** (hover) → 확인 모달 → 업무항목 + 관련 세션 삭제
- `+ 업무 추가` → 요일 탭 기반 드로어

**업무 추가 드로어:**
- 요일 탭(월~금) + 업무 카드 (프로젝트명 / 시작일·종료일 / 업무유형 / 메모)
- 고정 유형: 반복 요일 토글(월~금) 추가 표시
- `+ 업무 추가` 점선 버튼 / `저장하기` 버튼

**업무항목 데이터 구분:**
- **업무요청 수락 항목** (`sourceRequestId` 존재): 클라이언트 프로젝트명, 외부 참여자
- **직접 추가 항목** (`sourceRequestId` 없음): 내부 업무, 참여자 본인만

### 업무 상세/수정 슬라이드 패널 (500px)

- **업무요청 수락 항목** (`sourceRequestId` 존재):
  - `업무요청` 파란 배지 표시
  - 업무명·업무유형·시작일·마감일 수정 불가 (disabled + opacity:0.5)
  - 참여자 목록 표시
- **직접 추가 항목** (`sourceRequestId` 없음):
  - 배지 미표시, 모든 필드 수정 가능, 참여자 미표시
- 작업세션 목록 (카테고리 배지 없음)
- 변경 시만 `저장하기` 활성화

### col2: 오늘 할 일 (Daily Todo)

- **기본 날짜**: 오늘 (자동 설정)
- 시작시간 오름차순 플랫 목록, 시간 미입력 세션은 맨 아래
- 날짜 전환: Weekly Tasks 날짜 클릭 또는 날짜 피커
- **세션 추가**: 업무명 클릭 → 최상단 인라인 입력 → Enter → 맨 아래 배치
  - 카테고리 선택 없음 (제거됨)
- 세션 제목 수정: 더블클릭 → 인라인 input
- 완료 토글: 체크박스 (파란 체크 + 취소선)
- 시간 입력: 4자리 → HH:MM 자동 포맷
- 세션 삭제: hover → 확인 모달 → 물리 삭제
- 진행바: 완료/전체 진척률

### col3 상단: 오늘의 현황 (KPI)

- 1x2 그리드: 작업시간(파랑) / 세션 완료율(초록)
- 선택 날짜 기준 계산

### col3 하단: 업무 요청

- **클라이언트명 · 프로젝트명** 한 줄 통합 + 프로세스명 + 기간 + 수락/거절
- **수락** → 폼 모달 (업무항목명 + 기간 + 업무유형 + 프로세스명 + 작업 날짜)
- **거절** → 사유 선택 모달 (고정 목록 + 기타)
- 주간 자동 리셋

## 기타 탭

### Calendar — 월간/주간 캘린더
- `[전체 | 나]` 필터 + `[월간 | 주간]` 토글
- 비반복 업무 바/칩 + 반복 업무 dot 리스트
- 업무 클릭 → 상세 패널

### Team Status — 팀원 업무 현황
- 팀원별 카드 그리드 + KPI 4종 + 업무배정 현황
- `+ 업무요청 전송` (Manager/Owner만)

### Meeting Room — 녹음 + 회의록
- 녹음 → 저장 모달 → 회의 목록 + 상세 3탭 (AI/스크립트/액션아이템)

### Leave Management — 연차 관리
- KPI 3종 + 신청/취소/승인·반려

### My Page — 직원 대시보드
- 프로필 + AI 업무 분석 + 참여 회의 + 미니 캘린더
