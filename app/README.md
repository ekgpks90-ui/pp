# WorkFlow Hub Home Prototype

정적 HTML/CSS/JS로 구현한 Home 화면 인터랙티브 프로토타입.

## 실행

브라우저에서 `index.html`을 열면 된다.

## 레이아웃

```
[다크 사이드바 220px] | [메인 콘텐츠]
                           ├─ 헤더 (날짜 / 검색·알림·프로필)
                           └─ 3컬럼 그리드 (equal width)
                               ├─ 이번 주 업무
                               ├─ 오늘 할 일
                               └─ 오늘의 현황 + 업무 요청
```

## 디자인 시스템

- **폰트**: DM Sans (UI) / DM Mono (KPI 숫자)
- **사이드바**: `#0e0e16` 배경, 활성 탭 inset 좌측 보더
- **CSS 변수**: `--bg`, `--surface`, `--surface-muted`, `--line`, `--line-soft`, `--text`, `--muted`, `--soft`, `--blue`, `--blue-soft`, `--shadow`, `--shadow-lg`, `--radius`, `--radius-sm`
- **모달/오버레이**: `backdrop-filter: blur` 적용

## 구현된 기능

### 페이지 네비게이션
- 사이드바 Home / Calendar / Team Status / Meeting Room / Leave Management / My Page 클릭 시 페이지 전환
- 활성 탭 강조 (active 클래스 동적 관리)

### 공통
- 헤더: 주간 날짜 필터(좌) / 검색 인풋 · 알림 팝오버 · 프로필(우)
  - 주간 필터: `< 2026 06.08 ~ 06.14 이번 주 >` 형태, 화살표로 주 단위 이동
  - 검색: 헤더 내 인라인 인풋박스 (포커스 시 확장)
  - 알림 팝오버: `알림` 탭만 표시, 읽음/안읽음 표시, 전체 읽음 처리, 개별 클릭 읽음 처리
  - 알림 아이콘: 읽지 않은 알림 수 숫자 뱃지 표시 (99개 초과 시 `99+`), 모두 읽으면 뱃지 숨김
  - 알림 항목: 제목 + 본문 표시 (긴급/중요/일반 레벨 뱃지 없음)
  - 업무요청 알림: 제목 옆에 업무항목명 파란 칩으로 인라인 표시

### 이번 주 업무 (col1)
- 요일(월~일)별 카드로 그룹화 (7일)
- 오늘 카드는 연한 파란 배경, 나머지는 연한 그레이 배경
- 카드 내부: 요일명(bold) + 날짜(MM/DD) → 아이템 목록
- 각 아이템: 유형 아이콘 + 업무명 + 기간/반복 정보
  - 유형 아이콘: 핀(고정) · 빨간 원(긴급) · 그레이 원(일반)
- 카드 내 고정 → 긴급 → 일반 순 정렬, 마지막 고정업무 아래 구분선
- 지연 상태(end < today + 미완료 세션 존재)일 때만 `지연중` 배지 (고정업무 제외)
- **업무명 클릭** → 오늘 할 일 패널에 해당 업무의 세션 인라인 입력창 표시 (토글)
- **✏️ 편집 버튼** (hover 시 노출) → 업무 상세/수정 슬라이드 패널 오픈
- **🗑️ 삭제 버튼** (hover 시 편집 버튼 옆 노출) → 삭제 확인 모달 → 업무항목 + 관련 세션 전부 삭제
- `+ 업무 추가` → 요일 탭 기반 드로어 (우측 슬라이드, 500px)

**업무 추가 드로어:**
- 헤더: "Weekly Tasks" + X 닫기 버튼 (오른쪽 상단)
- 요일 탭: 월/화/수/목/금 — 균등 너비, 활성 탭 검정 배경/흰 글씨
- 탭별 업무 카드: 프로젝트명 / 시작일·종료일(2컬럼) / 업무유형 / 메모
- **고정 유형 선택 시**: 시작일·종료일(2컬럼) + 반복 요일 토글(월~금 버튼) 표시
- `+ 업무 추가` 점선 버튼으로 같은 날에 카드 추가 가능
- `저장하기` 전체폭 버튼으로 저장

**고정업무 반복 규칙:**
- `recurringDays` 배열로 반복 요일 지정 (1=월~5=금), 최소 1개 유지
- `end: null`이면 무기한 반복, 날짜 지정 시 종료일까지만 표시
- 기간 표시 대신 `매주 월·수·금` 형태로 표시

### 업무 상세/수정 슬라이드 패널 (500px)
- ✏️ 편집 버튼 클릭 → 우측 오버레이 슬라이드 패널 (배경 딤처리)
- X 버튼(오른쪽 상단) 또는 배경 클릭으로 닫기
- 드로어와 동일한 `dw-field` 폼 스타일로 통일
- 수정 가능 항목: 업무명 / 업무유형 / 마감일(일반·긴급) 또는 종료일+반복요일(고정) / 설명
- 시작일: 표시만 (수정 불가)
- 참여자 목록 / 오늘 작업세션 목록 표시 (오늘 날짜 기준)
- 변경 사항 있을 때만 `저장하기` 버튼 활성화 → 저장 시 workItems 업데이트

### 오늘 할 일 (col2)
- 오늘 해당하는 모든 세션을 **시작시간 오름차순** 플랫 목록으로 표시 (업무 그룹 없음, 시간 미입력 세션은 맨 아래)
- **세션 추가**: 이번 주 업무에서 업무명 클릭 → 오늘 할 일 **최상단**에 인라인 입력창 표시 (업무명 레이블 포함)
  - 카테고리 선택 + 세부항목명 입력 후 Enter → 세션 추가 (목록 맨 아래 배치), 입력창 닫힘
  - 세션 생성 시 시간은 비어 있음 (작업 완료 후 별도 입력)
- **세션 제목 수정**: 더블클릭 → 인라인 input 전환 → Enter 저장 / Esc·blur 취소(원래 제목 복원)
- 체크박스로 완료 토글 (완료 = 파란색 체크, 제목 취소선)
- 카테고리 컬러 배지 (디자인/기획/개발/운영/리서치/퍼블리싱)
- **시간 입력**: 숫자 4자리만 입력하면 HH:MM 자동 포맷 (예: `0923` → `09:23`)
  - Tab으로 시작→종료 시간 이동 (저장만, 재정렬 없음)
  - Enter로 저장 + 시간순 재정렬 / Esc로 취소(원래 값 복원)
  - 오늘 세션: 현재 시각 이후 입력 불가
  - 겹치는 시간 범위 입력 불가
  - 오류 시 인풋 위에 오류 툴팁 표시 후 원래 값 복원
  - 과거 날짜 세션은 제한 없음
- 세션 삭제 (hover 시 삭제 버튼 노출 → 확인 모달 → 물리 삭제)
- 진행바: 완료/전체 진척률 (그라디언트)

### 오늘의 현황 (col3 상단)
- 2×2 KPI 그리드, 고정 높이
- 작업시간(파랑) / 완료 세션(초록) / 남은 세션(주황) / 이번주 누적(보라)
- 값은 세션 state에서 실시간 계산

### 업무 요청 (col3 하단)
- 오늘의 현황 카드 아래 나머지 공간 전부 차지, 내부 스크롤
- 각 요청: 출처 팀 태그 + 설명 + 요청자/마감 + 수락/거절 버튼
- **수락** → 업무항목 폼 모달: 요청 원문 + 업무항목명·유형·시작일·마감일 자동 채움 → Weekly Tasks 추가
  - 업무항목명·업무유형은 고정값으로 수정 불가 (읽기 전용)
- **거절** → 사유 선택 모달 (고정 목록 + 기타 상세) → `거절됨` 배지 표시 + 버튼 숨김

### Leave Management 탭 (직원용)
- 헤더: 페이지 제목 + `+ 연차 신청` 버튼
- KPI 3종: 총 연차 / 사용 연차 / 잔여 연차
- 탭 바: 내 연차(대기 중) / 이력(처리 완료)
- 취소 버튼: 승인 대기 + 시작일 미도래 조건에서만 노출
- 연차 신청 모달: 유형(종일/오전 반차/오후 반차) + 시작일/종료일 + 사유 + 사용 일수 미리보기

### Meeting Room 탭 — 녹음 + 회의록

**녹음 플로우:**
1. 마이크 버튼 클릭 → 녹음 시작 (파형 애니메이션 + 빨간 타이머)
2. 정지 버튼 클릭 → 회의 저장 모달 오픈
   - 팀 선택 / 회의 유형 / 날짜 / 제목 / 내용
   - **참석자**: 이름 검색 드롭다운 → 선택 시 파란 칩으로 표시 (박스 하단), 칩 × 버튼으로 제거
   - **액션아이템**: 할 일 + 마감일 + 담당자(이름 검색 드롭다운 동일 방식) 행 추가
3. 저장 시 회의카드 목록 최상단에 추가

**회의 목록:**
- 팀 필터 탭: 전체 / 팀별 (동적 생성, 팀별 건수 표시)
- 회의 검색: 제목·유형·요약 포함 검색
- 회의 카드: 팀 태그 + 유형 배지 + 날짜/작성자 + 제목 + 요약 + 시간·참여자 아바타(전원 표시)·액션 건수
- **카드 hover** → 우상단 삭제 버튼 노출 → 삭제 확인 모달 → 회의록 삭제

**회의록 상세 패널 (우측 슬라이드, 3탭):**
- **AI 분석**: 주요 내용 + 주요 논의 목록
- **스크립트**: 타임스탬프 + 발화자 + 내용 라인
- **액션아이템**: 체크박스(완료 토글) + 담당자·마감일 + `→ 추가` 버튼 (담당자가 현재 로그인 유저일 때만 표시) → 이번 주 업무에 등록
- 기존 저장 회의록 8건 전체에 AI 분석·스크립트·액션아이템 데이터 포함

### Team Status 탭
- 팀원별 이번 주 업무항목 카드 그리드
- 각 카드: 아바타 + 이름 + 역할 + 업무항목 목록(유형 아이콘 + 항목명) + 완료율 진척바
- 상단 KPI: 팀 전체 완료율 / 지연 건수 / 업무 요청 현황
- 업무 요청 섹션: 팀 내 수신 요청 목록 (수락/거절 처리)

### Calendar 탭
- 카드 기반 디자인 (다른 탭과 일관성 통일)
- 헤더 우측 컨트롤: `[전체 | 나]` 탭 (데이터 필터) + `[월간 | 주간]` 토글 (뷰 방식) + `< >` 이동
- 요일 행: 일~토, 일요일 빨간색·토요일 파란색

**필터·뷰 조합 동작:**
| | 전체 | 나 |
|---|---|---|
| **월간** | 팀 전체 비반복 업무 → 상태별 가로 바 | 내 비반복 업무 → 유형별 가로 바 + 반복 업무 → 세부항목 dot 리스트 |
| **주간** | 팀 전체 비반복 업무 → 상태별 칩 (이번 주 시작일 기준 1개) | 내 비반복 업무 → 칩 + 반복 업무 → 세부항목 dot 리스트 |

**반복 업무 (세부항목) 표시:**
- 유형 아이콘: 핀(고정) · 빨간 원(긴급) · 회색 원(일반)
- 완료된 업무: 텍스트 취소선 + 흐리게 표시
- 클릭 불가 (상세 패널 열리지 않음)
- `recurringDays` 배열 있는 항목 전체가 대상 (type 무관)

**업무 상세 패널 (우측 슬라이드, 500px):**
- 업무항목 클릭 → 패널 오픈 (제목 좌측, X 버튼 우측)
- 참여자별 작업세션 그룹 표시 (아바타 + 이름 + 세션 목록)
- 아웃풋/리소스: 링크 URL 입력 + 파일 업로드 (각각 별도 인풋)
  - 업로드한 사람 이름 표시, 본인 업로드만 삭제 가능
  - hover 시 삭제 아이콘 노출 → 확인 모달 → 삭제

### My Page 탭 (직원용 대시보드)

**레이아웃:** 3컬럼 그리드 (180px 프로필 | 1fr 분석 | 380px 캘린더)

**KPI 바 (상단 5종):**
- 총 세션 수 / 진행 중 업무(업무항목) / 평균 작업시간 / 이번 달 작업시간 / 완료율

**프로필 카드 (좌측, 세로 꽉 채움):**
- 아바타 이니셜 + 이름 + 역할 뱃지
- 소속 팀 / 권한 / 입사일

**업무 분석 (센터 컬럼, 상→하 순서):**
1. **업무 패턴 요약 (AI)**: 이번 달 세션 기반 6가지 인사이트 (2열 그리드)
   - 가장 많이 수행한 업무 / 가장 많은 시간이 투입된 업무
   - 반복 업무 패턴 / 생산성이 높은 요일 / 생산성이 낮은 요일 / 업무 집중 시간대
2. **카테고리별 작업시간**: 가로 바 차트 (카테고리명 + 바 + 비율 + 시간)
3. **참여한 회의**: 소속 팀 기준 회의 목록 (유형 뱃지 + 제목 + 날짜 + 소요시간 + 참여인원)

**캘린더 + 세션 목록 (우측 컬럼):**
- 월간 미니 캘린더: 이전/다음 월 이동, 세션 있는 날짜 파란 점, 연차 날짜 주황 점
- 날짜 클릭 → 해당 날짜 작업세션 목록 표시 (카테고리 배지 + 세션명 + 소요시간)
