# Leave Management Wireframe

## 목적

연차 신청, 승인, 반려, 취소, 이력, 현황을 관리한다.

## 기본 레이아웃

```txt
┌────────────────────────────────────────────────────────────────────┐
│ Leave Management                                [+ 연차 신청]     │
├────────────────────────────────────────────────────────────────────┤
│ ┌────────────┬────────────┬────────────┐                         │
│ │ 총 연차    │ 사용 연차   │ 잔여 연차   │                         │
│ └────────────┴────────────┴────────────┘                         │
├────────────────────────────────────────────────────────────────────┤
│ [내 연차] [팀 연차] [승인 대기] [이력]                            │
├────────────────────────────────────────────────────────────────────┤
│ 연차 목록                                                          │
│ 날짜 / 유형 / 신청자 / 상태 / 승인자 / 액션                        │
└────────────────────────────────────────────────────────────────────┘
```

## 연차 신청 모달

```txt
┌─────────────────────────────┐
│ 연차 신청                   │
├─────────────────────────────┤
│ 유형                         │
│ ○ 종일 연차                  │
│ ○ 오전 반차                  │
│ ○ 오후 반차                  │
│                              │
│ 사용 날짜 [YYYY-MM-DD]       │
│ 신청 사유 [              ]   │
│                              │
│ [취소] [신청]                │
└─────────────────────────────┘
```

## 정책 메모

- 시간차는 MVP 제외이다.
- Member 신청은 Manager가 승인한다.
- Manager 신청은 Owner가 승인한다.
- 연차 시작 전까지만 취소할 수 있다.
- Calendar에는 연차를 표시하지 않는다.
- My Page에는 총/사용/잔여 연차 현황을 표시하지 않는다.
