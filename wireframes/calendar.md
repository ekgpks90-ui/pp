# Calendar Wireframe

## 목적

진행 중인 프로젝트의 단계별 일정을 Gantt 타임라인으로 조회한다.

## 기본 레이아웃

```txt
┌────────────────────────────────────────────────────────────────────┐
│ Calendar                                      [< 2026년 6월 >]    │
├──────────────────────┬─────────────────────────────────────────────┤
│ 진행 중 프로젝트     │ 모바일 앱 아이콘 세트 리뉴얼               │
│                      ├────┬────┬────┬────┬────┬────────────────── │
│ ● 모바일 앱 아이콘   │ 1  │ 5  │ 10 │ 15 │ 20 │ 25               │
│   세트 리뉴얼        ├────┴────┴────┴────┴────┴────────────────── │
│                      │ IA 설계     [이지혜 ████████]               │
│ ● 디자인 시스템 v2   │ IA 설계     [최유진 ██████]                 │
│                      │ 시안 제작   [이지혜 ████]                   │
│ ● 5월 브랜드 캠페인  │ 최종 확정   [이지혜 ██]                     │
│                      │                                             │
└──────────────────────┴─────────────────────────────────────────────┘
```

## 구성 요소

### 좌측 사이드바
- 진행 중 프로젝트 목록 (stepAssignees가 1개 이상 설정된 것만 표시)
- 클릭 시 우측에 해당 프로젝트 타임라인 표시

### 우측 Gantt 타임라인
- 상단: 선택된 프로젝트명 + 월 이동 컨트롤 (< 월 >)
- 헤더: 날짜 그리드 (일 단위)
- 행: 프로세스 단계별 / 담당자별 분리
  - 한 단계에 담당자 2명 이상: 담당자별 행 분리 (첫 행에 단계명, 이후 행은 공백)
- 바: 시작일~마감일을 해당 월 기준 % 위치로 표시 (담당자별 컬러 구분)

## 표시 정책

- 표시: 프로젝트 단계별 Gantt 바 (assignmentRequest 기반)
- 미표시: 연차, 작업세션, 회의 일정, 단순 업무항목
- Calendar는 업무 생성 화면이 아니다.
