# My Page Wireframe

## 목적

개인의 업무 기록과 생산성 데이터를 조회하고 분석한다.

## 기본 레이아웃

```txt
┌────────────────────────────────────────────────────────────────────┐
│ My Page                                                            │
├────────────────────────────────────────────────────────────────────┤
│ 계정 정보                                                          │
│ 이름 / 소속 팀 / 권한 / 입사일                                     │
├────────────────────────────────────────────────────────────────────┤
│ [작업 히스토리] [생산성 분석] [AI 업무 분석]                       │
├────────────────────────────────────────────────────────────────────┤
│ 작업 히스토리                                                      │
│ ┌──────┬──────┬──────┬──────┬──────┬──────┬──────┐               │
│ │ Mon  │ Tue  │ Wed  │ Thu  │ Fri  │ Sat  │ Sun  │               │
│ ├──────┼──────┼──────┼──────┼──────┼──────┼──────┤               │
│ │ 적정 │ 과부하│ 연차 │ 부족 │ 적정 │      │      │               │
│ └──────┴──────┴──────┴──────┴──────┴──────┴──────┘               │
│                                                                    │
│ 선택 날짜 작업세션 목록                                            │
└────────────────────────────────────────────────────────────────────┘
```

## 생산성 분석

```txt
┌────────────────────────────────────────────────────────────────────┐
│ 월별 작업시간 추이        완료 작업 추이                           │
│ [라인 차트]               [막대 차트]                              │
│                                                                    │
│ 카테고리별 작업 비율      업무 집중도                              │
│ [도넛 차트]               [시간대 차트]                            │
└────────────────────────────────────────────────────────────────────┘
```

## AI 업무 분석

```txt
┌────────────────────────────────────────────────────────────────────┐
│ 이번 달 업무 패턴 요약                                             │
│                                                                    │
│ - 가장 많이 수행한 업무                                            │
│ - 가장 많은 시간이 투입된 업무                                     │
│ - 반복 업무 패턴                                                   │
│ - 생산성이 높은 요일                                               │
│ - 생산성이 낮은 요일                                               │
│ - 업무 집중 시간대                                                 │
└────────────────────────────────────────────────────────────────────┘
```

## 정책 메모

- 총 연차, 사용 연차, 잔여 연차 현황은 표시하지 않는다.
- 연차 신청/승인/이력 관리는 Leave Management에서 한다.
- 작업 히스토리 캘린더의 연차 상태 표시는 가능하다.
