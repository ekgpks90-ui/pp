# Wireframes

WorkFlow Hub MVP 화면의 저해상도 와이어프레임을 보관한다.

목적은 시각 디자인 전에 화면 구조, 정보 우선순위, 사용자 흐름을 합의하는 것이다.

## 화면 목록

- [Home](home.md)
- [Task Request](task-request.md)
- [Calendar](calendar.md)
- [Meeting Room](meeting-room.md)
- [Leave Management](leave-management.md)
- [My Page](my-page.md)
- [Category Management](category-management.md)

## 공통 원칙

- 업무 입력과 조회가 빠른 SaaS 운영 도구 화면을 기준으로 한다.
- Home은 실제 작업 화면이며 랜딩 페이지가 아니다.
- Weekly Tasks는 업무항목 영역이다.
- Daily Todo는 작업세션 영역이다.
- Calendar는 조회 화면이며 업무 생성 화면이 아니다.
- My Page에는 총/사용/잔여 연차 현황을 표시하지 않는다.
- 권한이 없는 액션은 숨기거나 비활성화한다.
