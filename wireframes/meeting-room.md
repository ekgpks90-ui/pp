# Meeting Room Wireframe

## 목적

회의를 생성하고 AI 회의록, 액션아이템, 업무요청 전환을 관리한다.

## 목록 화면

```txt
┌────────────────────────────────────────────────────────────────────┐
│ Meeting Room                                      [+ 회의 생성]    │
├────────────────────────────────────────────────────────────────────┤
│ Filters: [회의 유형] [작성자] [참석자] [날짜] [검색]              │
├────────────────────────────────────────────────────────────────────┤
│ 회의 목록                                                          │
│ ┌───────────────────────────────────────────────────────────────┐  │
│ │ 회의 제목 / 유형 / 작성자 / 작성일 / 참석자 수                │  │
│ └───────────────────────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────────────────────┘
```

## 상세 화면

```txt
┌────────────────────────────────────────────────────────────────────┐
│ 회의 제목                                      [참석자 관리]       │
│ 회의 유형 / 작성자 / 작성일 / 소속 팀                              │
├────────────────────────────────────────────────────────────────────┤
│ [AI 요약] [스크립트] [액션아이템]                                  │
├────────────────────────────────────────────────────────────────────┤
│ AI 요약 탭                                                         │
│ - 회의 요약                                                        │
│ - 핵심 논의사항                                                    │
│                                                                    │
│ [수정] [저장]                                                      │
└────────────────────────────────────────────────────────────────────┘
```

## 액션아이템 탭

```txt
┌────────────────────────────────────────────────────────────────────┐
│ 액션아이템                                                        │
├──────────────┬──────────┬──────────┬──────────────┬──────────────┤
│ 업무명       │ 담당자   │ 마감일   │ 상태         │ 액션         │
├──────────────┼──────────┼──────────┼──────────────┼──────────────┤
│ 시안 검토    │ 김디자   │ 06-20    │ 진행 예정    │ 업무요청 전환 │
└──────────────┴──────────┴──────────┴──────────────┴──────────────┘
```

## 정책 메모

- 액션아이템 상태는 회의록 내부 상태이다.
- 액션아이템은 업무항목이 아니다.
- 액션아이템을 업무요청으로 전환해도 업무항목은 자동 생성되지 않는다.
- 업무요청 수락 후 Weekly Tasks에 추가해야 업무항목이 된다.
