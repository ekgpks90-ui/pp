6. My Page
목적
개인의 업무 기록과 생산성 데이터를 조회하고 분석하는 기능이다.

6-1. 개인 작업 히스토리
목적
과거 업무 기록을 조회한다.

표시 방식
월간 캘린더

표시 상태
	•	업무 과부하
	•	업무 적정
	•	업무 부족
	•	연차

연차 표시 정책
	•	작업 히스토리 캘린더에서 연차 상태 표시는 가능하다.
	•	총 연차, 사용 연차, 잔여 연차 현황은 표시하지 않는다.
	•	연차 신청, 승인, 이력, 총/사용/잔여 연차 현황은 Leave Management에서 관리한다.

날짜 선택
선택한 날짜의 작업세션 조회

표시 항목
	•	업무명
	•	세부업무명
	•	업무카테고리
	•	작업시간
	•	소요시간

6-2. 개인 성과 요약
목적
개인의 업무 수행 현황을 요약한다.

표시 항목
	•	완료 작업세션 수
	•	진행중 작업 수
	•	평균 작업시간
	•	월간 작업시간
	•	카테고리별 작업 비율

6-3. 생산성 분석
목적
업무 데이터를 기반으로 생산성 현황을 분석한다.

표시 항목
	•	월별 작업시간 추이
	•	완료 작업 추이
	•	카테고리별 작업 비율
	•	업무 집중도

차트
	•	막대 차트
	•	라인 차트
	•	도넛 차트

6-4. AI 업무 분석
목적
업무 데이터를 분석하여 인사이트를 제공한다.

분석 항목
	•	가장 많이 수행한 업무
	•	가장 많은 시간이 투입된 업무
	•	반복 업무 패턴
	•	생산성이 높은 요일
	•	생산성이 낮은 요일
	•	업무 집중 시간대

AI 요약
이번 달 업무 패턴을 자연어로 요약한다.

6-5. 계정 정보
표시 항목
	•	이름
	•	소속 팀
	•	권한
	•	입사일
