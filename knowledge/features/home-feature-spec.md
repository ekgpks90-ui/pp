# Home

## 목적
오늘 수행해야 할 업무를 확인하고 업무를 계획 및 기록하는 메인 화면이다.

---

## 구성 요소
- KPI 위젯
- Weekly Tasks
- Daily Todo
- 업무요청 위젯

---

## 1. KPI 위젯

### Member
- 오늘 작업시간
- 오늘 완료 작업세션
- 오늘 남은 작업세션
- 이번주 누적 작업시간

### Manager (Member KPI 포함)
- 지연중 업무
- 긴급업무
- 응답 대기
- 과부하 직원

### Owner (Manager KPI 포함)
- 진행중 업무
- 조직 현황

## 2. Weekly Tasks

### 목적
주간 업무항목을 관리한다.

### 생성 경로
- 직접 생성 (`+ 업무 추가` 버튼)
- 업무요청 수락
- 회의 액션아이템

### 필수 입력값
- 업무명
- 시작일
- 마감일
- 업무유형: `일반` / `고정` / `긴급`

### 고정업무 반복 규칙
| 항목 | 내용 |
|------|------|
| 반복 요일 | 월~금 중 복수 선택 필수 |
| 종료일 | 선택사항. 없으면 무기한 반복 |
| 표시 조건 | 해당 주에 반복 요일이 포함되고 시작일 이후이면 표시 |
| 완료 처리 | 요일별 독립 체크. 매주 초기화 |
| 지연 배지 | 고정업무에는 표시하지 않음 |

### 표시 정책
- 업무항목 자체에 배경 컬러 적용하지 않음
- 오늘 요일 영역에만 컬러 강조
- 업무항목 정렬 순서: 고정 → 긴급 → 일반
- 업무유형 마커: 고정(고정 마커) / 긴급(긴급 점) / 일반(일반 점)
- 고정업무 아래에만 구분선 표시

### 지연중 조건
- 마감일 경과
- 미완료 작업세션 존재

## 3. Daily Todo

### 목적
선택한 업무항목의 작업세션을 관리한다.

### 날짜 선택
- 기본값: 오늘
- 다른 날짜 선택으로 미리 작성 가능

### 필수 입력값
- 세부업무명
- 날짜

### 선택 입력값
- 시작시간
- 종료시간

### 검증 규칙
- 시작시간 < 종료시간 필수
- 0분 작업 생성 불가

### 업무요청 수락 시 자동 배치
- Weekly Tasks: 프로젝트명·시작일·마감일 자동 추가
- Daily Todo: 수락 시 직원이 날짜 선택 → 선택한 날짜에 세부업무항목 자동 생성

---

## 4. 업무요청 위젯

### 표시
- 신규 요청 수 / 수락 대기 수

### 기능
- 수락
- 거절 (거절 시 사유 입력 필수)
