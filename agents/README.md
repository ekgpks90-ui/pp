# Agents

WorkFlow Hub 문서를 다루는 AI/작업자 역할별 지침을 보관하는 공간이다.

공통 지침은 루트의 [CLAUDE.md](../CLAUDE.md)를 따른다.

## 임원진 구성

- [CEO](ceo.md): 제품 방향성과 MVP 범위 관리
- [CPO](cpo.md): 기능 우선순위와 사용자 플로우 관리
- [CTO](cto.md): 데이터 모델, API, 권한, 구현 정책 관리
- [COO](coo.md): 실제 조직 운영 정책 검토
- [CDO](cdo.md): 업무 데이터, KPI, 분석 관점 관리
- [QA Lead](qa-lead.md): 완료 기준, 예외 케이스, 권한 테스트 검수
- [UX Lead](ux-lead.md): 화면 구조, 입력 흐름, 사용성, 정보 위계 전담
