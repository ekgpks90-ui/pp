# WorkFlow Hub Design Notes

## 제품 성격

운영 분석 SaaS이다. 화면은 업무 입력과 데이터 조회를 빠르게 반복할 수 있어야 한다.

## UI 방향

- 대시보드보다 업무 기록과 분석 흐름을 우선한다.
- Weekly Tasks는 업무항목 계획 영역이다.
- Daily Todo는 작업세션 입력 영역이다.
- Home은 오늘 해야 할 일과 기록할 작업을 한 화면에서 처리하는 중심 화면이다.
- Calendar는 업무 생성 화면이 아니라 일정 조회 화면이다.
- My Page는 개인 업무 데이터 분석 화면이며, 총/사용/잔여 연차 현황은 표시하지 않는다.
- My Page 작업 히스토리 캘린더에는 연차 상태 표시만 허용한다.

## 핵심 화면

### Home

- KPI 위젯
- 업무요청 위젯
- Weekly Tasks
- Daily Todo
- 팀원 업무 현황

### Calendar

- Team 필터
- My 필터
- 업무항목 일정
- 회의 일정

### Meeting Room

- 회의 생성
- 녹음
- AI 회의록
- 액션아이템
- 업무요청 전환

### Leave Management

- 연차 신청
- 연차 승인/반려
- 연차 취소
- 연차 이력
- 연차 현황

### My Page

- 작업 히스토리
- 생산성 분석
- AI 업무 분석
- 계정 정보
