# WorkFlow Hub Plan

## 목적

디자인 조직의 업무 수행 데이터를 수집하고 분석할 수 있는 MVP를 만든다.

## MVP 범위

- Home
- Weekly Tasks
- Daily Todo
- Task Request
- Calendar
- Meeting Room
- Leave Management
- My Page
- Category Management
- Notification

## MVP 제외

- Business Dashboard
- 매출 관리
- 프로젝트 수익성
- 현금 흐름
- ERP 기능
- 타이머 기능
- 상위/하위 프로젝트 구조
- 업무항목 완료 상태
- 시간차
- 거절 사유 관리 화면

## 구현 우선순위

1. 핵심 데이터 모델: [기획용 데이터 모델](knowledge/data-model.md) 기준
2. Home: KPI, Weekly Tasks, Daily Todo
3. 업무요청: 생성, 수락, 거절, Weekly Tasks 추가
4. 권한 정책: Owner, Manager, Member
5. Calendar: 업무항목/회의 일정 조회
6. Leave Management: 신청, 승인, 반려, 취소
7. Meeting Room: 회의록, AI 요약, 액션아이템, 업무요청 전환
8. My Page: 작업 히스토리, 생산성 분석, AI 업무 분석
9. 알림과 변경 이력 저장
